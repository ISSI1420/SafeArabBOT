## TON Escrow Telegram Bot

بوت وساطة تلغرام كامل يدعم TON وUSDT-TON – صفقات – ضمان – نزاع – إحالة – قائمة نصابين – إدارة أدمن – تقييم – تحرير تلقائي من المحفظة – جاهز للنشر على Render.

### التشغيل

1. انسخ ملف `.env.example` إلى `.env`، أدخل توكنك وباقي البيانات.
2. شغل:
    ```
    pip install -r requirements.txt
    python3 -m src.main
    ```
3. رفع إلى Render.com كـ worker:
    ```
    worker: python3 -m src.main
    ```

**ملحوظة:**
- أدخل seed لمحفظة جديدة خاصة بالبوت (ولا تعرضه أو تشاركه أبداً!)
- كل منطق الصفقات والإيداع والسحب أوتوماتيك/إداري وأكثر – جاهز للتطوير!
